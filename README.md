# Mithril with TypeScript Vite Starter

[Vite](https://vitejs.dev/) starter template to scaffold a new [Mithril](https://mithril.js.org/) with TypeScript project using https://github.com/ArthurClemens/mithril-ts-vite-starter

## npm scripts

* `npm run dev` - Starts the development server at port 3000
* `npm run build` - Builds the application
* `npm run preview` - Serves the build files locally at port 5000
